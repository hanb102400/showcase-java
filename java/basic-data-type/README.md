
## 基本数据类型

数据类型

| 类型    | 存储大小 | 范围                 |
| ----- | ---- | ------------------ |
| boolean  |  | true/false         |
| char  | 2个字节 |         |
| byte  | 1个字节 | -128 ~127          |
| short | 2个字节 | -32768~32767       |
| int   | 4个字节 | - 2^32 ~ (2^31-1)  |
| long  | 8个字节 | - 2^64 ~ (2^64 -1) |
| float  | 4个字节 | -128 ~127    |
| double | 8个字节 | -32768~32767 |
